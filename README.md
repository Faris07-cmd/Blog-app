# Django Blog App (Work in Progress)

This is a **Django blog project** currently under development.  
The project currently includes minimal user authentication features, but most pages and functionality are not implemented yet.

---

## Features (Implemented so far)

- User registration page (not linked from main page)
- Basic user login/logout pages
- Django project structure with templates and static folders

---

## Planned Features

- Main blog page
- Linking registration/login from homepage
- User profile page
- CRUD operations for blog posts
- Comments and likes system

---

## Getting Started

### Prerequisites

- Python 3.11+  
- pip  
- virtualenv (optional but recommended)
